# Retro Calculator

## To clone project:

git clone https://github.com/gmoraleslondono/retro-calculator.git

cd retro-calculator

code .

git init

npm i

## To install cypress:
npm i -D cypress

## To run integration test:

npx cypress open

### create a local sever:
npx http-server src/

open http://127.0.0.1:8080/
